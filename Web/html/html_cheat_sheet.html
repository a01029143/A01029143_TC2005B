El tag "< p >" se utiliza para definir el apartado de un párrafo en una página web para así organizar el contenido.

Ejemplo:

Tag 3: "< h1 >", "< h2 >", "< h3 >", "< h4 >", "< h5 >" y "< h6 >"

El tag "< h(1-6) >" se utiliza para crear subtitulos en páginas web, cada uno de los 6 crea un subtitulo de diferente tamaño, con el h1 siendo el más grande.

Ejemplo:

Este es el subtitulo h1

Este es el subtitulo h3

Este es el subtitulo h4

Este es el subtitulo h5
Este es el subtitulo h6

Tag 24: "< ul >"

El tag "< ul >" se utiliza para crear una lista que no tiene un orden específico. (Usa viñetas o puntos)

Ejemplo:

  • Martes
  • Lunes
  • Sábado

Tag 25: "< ol >"

El tag "< ol >" se utiliza para crear una lista con un orden específico. (Usa números, letras o números romanos)

Ejemplo:

  1. Despertarse
  2. Levantarse
  3. Tender la cama
